# 竞品分析

本目录专门沉淀知识选择 Feed 的竞品、反面案例和产品启示。

## 竞品格局总览

| 产品/类型 | 已经满足什么 | 对本项目的启发 |
|---|---|---|
| 知乎 / Quora | 高密度经验、问答、争论、案例 | 价值在真实经验和不同立场，但筛选成本高 |
| 知乎直答 / AI Search | 综合回答、搜索式问答 | 适合用户已有问题，不适合无目的刷 |
| Duolingo | 短任务、反馈、留存、游戏化 | “先做动作，再给反馈”比单纯阅读更容易留存 |
| Brilliant | 互动题、逐步理解、挑战感 | 互动学习成立，但更偏确定性 STEM 训练 |
| Headway / Blinkist | 短知识、微学习、订阅 | 纯摘要容易变被动阅读，缺少选择参与感 |
| Quizlet / NotebookLM | 从材料生成 flashcards/quizzes | 生成题目会变成基础能力，不能作为长期差异化 |
| Perplexity / ChatGPT Search | 即问即答、来源引用 | 是搜索入口，不是内容 feed 和判断训练入口 |
| ELYS / AI 社交 | AI 分身、AI 评论制造互动感 | AI 不能伪造真实他者确认，不能伪造用户评论和高价值用户活跃 |

## 关键反定位

1. 不做低配 Headway：不能只是把长内容压缩成摘要。
2. 不做低配 Quizlet：不能只是把内容变成单选题。
3. 不做 AI 搜索入口：不要让用户必须先有明确问题。
4. 不做 AI 伪互动社区：AI 可以辅助整理、分类、推荐，但不能假装是真人评论、点赞或高价值用户。
